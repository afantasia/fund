라온보드 및 세팅환경 작업
git 다운로드
1. ssh : git clone https://github.com/afantasia/fund.git 폴더명(ex: laonboard) ( www 에서) <br>
2. git이 다운로드된 이후 설치된 웹사이트에 들어가 설치(관리자 정보 입력)
3. ssh : php artisan migrate  
4. ssh : php artisan db:seed

<b>위 형태로 세팅하여 작업할 예정입니다.</b><br>
